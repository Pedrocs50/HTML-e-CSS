Principais formatações

Negrito / Destaques

Nesta frase HTML está em Negrito, utilizando a função "bold"(B, não Semântica).

Nesta frase CSS está em Destaque, utilizando a função "strong"(strong, Semântica)

Itálico / Ênfase

Nesta frase JS está em Itálico, utilizando a função "i".

Nesta frase JS está sendo formatado em itálico com o auxílio do CONTROL + SHIFT + P.

Nesta frase temos um termo em Ênfase utilizando a tag EM(Semântica)

Texto marcado(marcador de texto)

Neste frase o texto está marcando com a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste paragrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto insirido

Podemos marcar um texto como inserido para dar uma Ênfase e indicar que ele foi adicionado depois.

Tem também a tag "u" que não é Semântica.

Texto sobrescrito

Para insirir coisas do tipo x20+3

Texto subscrito
